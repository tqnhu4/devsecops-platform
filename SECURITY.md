# Security Policy

## Reporting Security Issues

If you discover a security vulnerability:

1. Do not open a public issue
2. Report privately
3. Include reproduction steps

# Security Controls

This platform includes:

- Trivy scanning
- Cosign signing
- Kyverno enforcement
- Falco runtime security
- Network policies
- Readonly filesystem policies
